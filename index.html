Hugo+Github建立自己的个人博客

大家好啊！我是Parzival，这篇文章主要是记录了一下我这个萌新从零开始搭建自己的个人博客的步骤。 观前须知：本人也是超级超级小白，弄这个博客之前对前端几乎是一无所知，搜了好几个教程，前前后后花了十几个小时才弄好个人博客，写的这个教程出现差错也是正常的，欢迎各位指正～ 同时，如果你也是一枚萌新，那一定一定要做好心理建设，弄这个东西出错是很正常的，我当时看教程，觉得就这么六七步，轻轻松松两个小时包拿下，结果前前后后花了2天时间才弄好，期间真的是无数次红温😡项目重开了无数次。所以，请你不要被失败打击到，出现错误后就想办法去调试解决，解决不了的话就删库重开，没什么大不了的，加油！ 前期准备工作 个人电脑一台，并了解一些基本的命令行相关知识，同时电脑安装有git； Github账号一个； 参考资料查阅准备： 使用hugo+github-pages部署个人博客 hugo官方使用文档中文版 hugo官方使用文档英文版 hugo模版库 gh-pages如何关联自己的域名 Markdown如何插入图片 这篇文章的目的是实现一个比较完整的个人博客搭建，包括Github Action自动生成静态内容 并且发布以及gh-pages关联自己的域名等操作，如果你只是想搭建一个 简单的个人博客，那么可以不必详细阅读我的博文，第一个参考资料使用hugo+github-pages部署个人博客 会是一个更好的选择，通过学习那一篇博文，你可以用更简单的操作，更短的时间快速搭建一个个人博客 第一步：创建仓库 我已经默认读到这篇文章的你拥有了Github账户 那么第一步是创建两个Github仓库,一个用来存放自己的整个博客项目，它是私有的 我们称之为博客源仓库，或简称为源仓库，另一个仓库则用来发布自己的博客，我称之为public 仓库。 首先，我们来创建自己的源仓库： 仓库的名称随意，例如我给它命名为“MyBlog”，但请你最好不要把它命名为username.github.io 形式😂 接下来，请勾选Private，因为我希望我的博客源仓库仅自己可见，如果你不介意把它公开的话， 那么完全不需要创建两个仓库，可以参考id1自行完成1个仓库的创建以及后续流程 然后请勾选Add a Readme File，这样会自动把这个仓库的branch 初始化为Main，如果不这么做的话 你的仓库会是空的，后续添加新东西的时候，git应该会把默认branch初始化为Master，本质上二者没什么区别 只是我学到的教程都教我这么做，所以我也习惯这么做了😂 下面来创建第二个仓库，也就是我们用来发布博客的远程仓库。 注意！这里的仓库名称必须使用username.github.io的格式，其中，username就是图片中 左侧的“Owner”显示的内容，它是独一无二的，例如，我的username叫做Parzival-Chen, 所以我这里就应该填写parzival-chen.github.io，大小写本质上没什么区别，看你心情填写即可。 因为我已经创建过自己的这个仓库了，所以这里出现了报错，提示我“这个仓库已经存在”，如果你之前没有用这个名字创建过 仓库，那么这里应该是不会报什么错误的 第二步 安装Hugo 请参考hugo官方文档或者英文版里的安装步骤进行，例如，我使用的是MacBook，那么我 需要在命令行里输入brew install hugo,安装完成后，可以输入hugo version来检查自己 是否安装成功以及自己的hugo版本 第三步 本地创建博客项目文件夹 现在，我们需要挑选一个合适的风水宝地，建立自己的博客项目文件夹，例如，我随意地在桌面上 新建了一个文件夹，并命名为“Parzival Blog” 然后回到第一步中创建的博客源仓库，复制SSH链接 接着打开位于刚刚创建的本地博客项目文件夹的终端，windows用户请win+R然后输入cmd进入命令行， 接着cd “文件夹的路径”进入这个文件夹， 然后输入git init 把它初始化为一个git 仓库， ...

鼻中隔偏曲手术全流程分享

前情提要 省流版：若只关心手术流程可以跳过本节，直接前往下一节“术前检查” 我本人长期以来饱受鼻炎折磨，具体表现为：鼻子经常通气不顺畅， 会感觉呼吸不是很舒服，有点吃力；经常擤鼻涕（帮助我从小养成了随身带纸的好习惯）； 感冒后症状通常会比没有鼻炎的正常人更严重，并且鼻塞症状长时间难以消退。 所以隔三差五就会去医院的耳鼻喉科就诊。中药西药都吃过，但是感觉基本没什么效果， 唯一比较管用的就是喷雾类药物，例如雷诺考特橙色瓶，但是这种喷雾不能久用，可能会造成药物性鼻炎， 以及鼻黏膜损伤等后果。 2022年，高考后的暑假，在家乡的耳鼻喉医院又一次做检查，CT拍片检测出我存在鼻中隔偏曲， 通俗来说，鼻中隔就是鼻子中间的那部分软骨，我的软骨就有一部分偏向了鼻子右侧。 同时还检测出黏膜囊肿和双侧鼻甲肥大等问题。在我没有既往过敏史的情况下， 医生表示这些症状可能是鼻中隔偏曲引起的，建议手术治疗。（全麻手术） 但是我的父母对此抱有怀疑态度，他们本身性格就偏保守， 再加上家乡的那家比较出名的耳鼻喉医院是私立医院，有可能出现推销医疗的情况， 决定等我上大学后去成都的好医院再看看。 来到电科上学后，我到华西医院上锦医院的耳鼻喉科挂号就诊，还花大价钱挂了一个一级专家， 97元的挂号费让我十分肉疼。医生检查了一下我的鼻子后就让我去做CT检查， CT检查后，医生指着片子跟我说我有鼻中隔偏曲，建议我手术治疗。 但是当时正是疫情时期，我的家属不能前往四川陪护，这个手术也不是什么紧急治疗， 我就暂且把它搁置下了。 这一拖就拖到了2024年4月底，我不幸患上了流感，流感+鼻炎双鬼拍门， 给我折磨得痛不欲生，我就又挂了一个华西医院上锦医院的耳鼻喉科的号。 虽然说2022年在这里拍过CT，医生还是要求我再做一次CT检查，检查结果和上次基本一样， 医生强烈建议我手术治疗。但是我的父母还是反对手术，表示暑假回家再议。 暑期归乡后，我又挂号去家乡的耳鼻喉医院，商讨手术事宜。医生含糊地表示总费用大概在6K多人民币， 医保报销后自费3K多，术后需要住院4-5天。回家后，父母决定再带我去别的地方的大医院看一看， 就算做手术也想去大医院做，他们表示不太能信任家乡的那家耳鼻喉医院。 所以我又挂了北京大学第一医院耳鼻喉科的一位鼻科专家的号。不出意外，这位医生也建议我做手术治疗。 然后我就在这里排上了鼻中隔偏曲手术的号。感觉鼻科手术在这里相对来说没那么热门，医生说预计一周后就可以做。 术前检查与准备 需要进行常规的胸片，心电图，血液化验等检查，北大第一医院是三甲医院， 我一共花费接近900元，由于我是异地医保，这个门诊检查费需要我回到参保地报销， 所以现在我还不知道这个检查费能报销多少。 一周后，我再次来到北京，办理入院手续后，护士和管床医生对我进行了一些常规的检查， 包括身高体重，血压测量，修剪鼻毛和胡子（记得自己带上鼻毛修剪器和剃须刀）。 额外需要一提的是手术风险知情书，内容还是有点吓人的，做这个手术有鼻中隔穿孔、 空鼻症、塌鼻子、鼻腔黏膜脱落等等的风险。全身麻醉还有可能导致牙齿脱落，声带受损（因为到时候会有一根管子塞到我嗓子里）。 当时是给我吓了一跳，但是医生说我还比较年轻，这些风险发生的概率比较低。 最后我还是硬着头皮签了字😂。 手术当日 我是被安排到了当天的第二台手术，被要求起床后不吃不喝。然后10点被推到麻醉室， 我还像个好奇宝宝一样环视麻醉室，询问麻醉师我有没有可能牙齿脱落，麻醉师微微一笑， 说“你不会知道这些的，在这之前你就会被麻醉了”。 全麻手术麻醉很快，静脉注射+吸入麻醉，麻醉剂推进来的时候感觉手腕非常冰， 我算是第一次体验到了什么是“手脚冰凉”，然后几乎是同一时间，我立刻就感受到了困意，大约五秒后， 我就两眼一黑倒了下去。 我醒来的时候大约是12点，所以麻醉+手术总耗时约2个小时。此时的我还不知道，真正的噩梦才刚刚到来。 PS：这一段是真的没什么好写的，毕竟全麻😂 术后48小时 手术后的48小时是最难顶的一段时间，因为我的两个鼻孔都被塞入了棉条（为了止血和塑形）， 我这两天全程都只能用嘴呼吸。这和我们平时用嘴呼吸是完全不一样的！ 主要的痛点有4点： 鼻腔可能会不停地渗血/水，这是正常情况，所以请务必带上很多很多抽纸， 最好买那种柔软的婴儿抽纸，防止使用过度频繁擦伤鼻子。 口腔会非常非常干燥，嘴唇，口腔，嗓子眼感觉一直像是沙漠一样，术后不到24小时， 我的嘴唇已经开裂了。我甚至还专门买了加湿器放到医院病房，的确是起到了一点作用， 但是买回来已经是术后第二天了，术后当天最难受的时候没有加湿器用😭。 这一点可能因地区而异，我是在北京住院的，比较干燥，可能南方的医院会好那么一点。 吃饭喝水有些许不适，因为鼻腔被完全堵死，吃饭喝水的时候嗓子的气管也被堵住了， 这个时候身体内外就会产生一些轻微的压强差，吞咽的时候就会有一点不适感， 这一点也是因人而异，我自己适应地还行，基本没有影响吃饭，但是有的患者甚至都只能吃流食， 吞咽一用力就会呛住。 晚上睡不着觉。这一点是最难绷的，我本人对睡觉环境的要求还是不低的， 所以这个鼻腔堵死+口腔干燥对我的影响非常大。再加上我平时就不习惯用嘴呼吸， 所以快睡着的时候就会把嘴闭上，然后被憋醒，真的是直接红温。每次都是我躺在床上用嘴呼吸， 然后困意袭来，我快要睡着了，就慢慢把嘴闭上了，然后呼吸被中断，直接憋醒，反复循环。 所以我在术后36小时完全是没睡过觉的，纯在坐牢。术后第二天赶紧向医生要了一点安眠药，第二天晚上才算是勉强睡了个觉。 所以，建议要做这个手术的你在手术前就找医生开上安眠药，以免自己术后坐牢。 出院与花费 术后48小时后，医生给我取掉了鼻腔里的两根棉条。这时候就会舒服很多， 鼻腔黏膜还是会水肿，永别了，牢笼！虽然此时仍然无法用鼻子呼吸，但是至少鼻腔内部和外部连通了， 口干舌燥的情况缓解了很多，整个人呼吸的体验一下子就不一样了。 ...
